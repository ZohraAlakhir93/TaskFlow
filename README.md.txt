# TaskFlow

## Description
TaskFlow est une application de gestion de tâches collaboratives développée dans le cadre du module de conception logicielle.

Cette application permet aux utilisateurs de :
- créer des tâches
- gérer des projets
- suivre l’avancement du travail
- recevoir des notifications

---

## Fonctionnalités principales

### Gestion des utilisateurs
- Création d’utilisateurs
- Attribution des rôles

### Gestion des projets
- Création de projets
- Ajout et suppression des membres

### Gestion des tâches
- Créer une tâche
- Modifier une tâche
- Supprimer une tâche
- Changer le statut d’une tâche

### Notifications
- Notification des utilisateurs lors des changements

---

## Design Patterns utilisés

### Singleton
Utilisé pour le service de notification afin de garantir une seule instance.

### Observer
Utilisé pour notifier automatiquement les utilisateurs lors des modifications.

### Factory
Utilisé pour créer différents types de tâches.
